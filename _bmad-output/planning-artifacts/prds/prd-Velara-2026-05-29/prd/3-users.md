# 3. Users

**Matt Maxwell** — Vitalief CIO. Primary skill author. Methodology certifier (second key in the two-key workflow). Approves data handling policy. Defines what qualifies as Vitalief-grade output.

**Vitalief consultants** — Primary operators of internal skills during active engagements. Invoke skills on behalf of clients or for internal use. Scoped to engagements they are assigned to.

**MA Technologies team** — Platform builder (Phase 1). Secondary skill author and technical validator (Phase 2). Holds the first certification key (technical quality).

**Client engagement leads** — Decide which client-facing skills are active for a given engagement. Grant or revoke client access to specific skills (Phase 2).

**Clients** — Invoke client-facing skills through an approved surface. Receive outputs. Cannot see skill internals at any point.
